# Changelog

## 0.1.0 (2026-03-11)

Initial release.

### Features

- **8 understanding dimensions**: personality, values, decision, aesthetics, communication, knowledge, prediction, boundaries
- **7-tier evolution system**: Stranger → Sketch → Understand → Sync → Mirror → Symbiosis → Transcend
- **Dual-write mechanism**: every signal updates both XP (quantitative) and Profile (qualitative)
- **3 AI providers**: OpenAI (gpt-4o-mini), Anthropic (claude-haiku), Ollama (llama3.1)
- **3 storage adapters**: Memory (zero-config), SQLite (persistent), Supabase (production)
- **Custom dimensions**: replace defaults with your own domain-specific dimensions
- **Signal hooks**: `onSignal` callback for real-time signal monitoring
- **User corrections**: highest-quality signals (25 XP each) from explicit user feedback
